# 依赖、数据与知识产权边界

## 当前离线 Demo

| 项目 | 状态 | 许可证/边界 |
|---|---|---|
| Python 标准库 | 必需 | Python Software Foundation License；不随仓库分发解释器 |
| `data/demo_cathodes.csv` | 必需 | 项目自建 illustrative demo 数据；不声称来自实验或论文，不允许用于科学决策 |
| Axiomatter LabOps 源码与 Skill | 必需 | Apache-2.0 |
| AgentTeams | 复赛集成目标 | 官方项目，Apache-2.0；当前未复制其源码或镜像到本仓库 |
| OpenAI 或其他模型 API | 可选增强 | 默认关闭；使用者自行遵守服务条款、费用和数据边界 |

## 提交披露

- 项目基于已有 Axiomatter 材料筛选原型继续开发；GOAI 新增部分为多 Agent 编排、Identity、Skill、Trace、审批闸门、审计、评测与竞赛材料。
- 不私下与其他参赛团队共享代码。若未来共同开发，按组委会规则合并队伍并保留贡献记录。
- 引入公开数据前记录来源 URL、版本、下载日期、许可证、允许用途、清洗步骤与哈希。
- 引入合作方数据前取得书面授权并隔离存储，不把受限数据提交到公开仓库。
- 商标、域名与正式项目名尚未检索清关；Axiomatter LabOps 仅为参赛工作名。
