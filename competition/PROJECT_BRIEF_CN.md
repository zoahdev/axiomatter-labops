# Axiomatter LabOps - GOAI Agent Infra 初赛底稿

## 一句话

把企业研发需求变成可审计、可回放、有人类安全闸门的多 Agent 执行证据链。

## 500 字以内作品简介

先进材料、生物医药与工业研发团队正在使用 AI 生成候选方案，但“谁依据什么数据做了什么判断、失败如何处理、实验由谁批准、结果能否复现”仍散落在聊天、脚本和表格中，导致 Demo 难以进入真实研发流程。Axiomatter LabOps 是基于 AgentTeams 设计的开源研发执行基础设施：Manager 将目标拆给需求、证据、候选、实验、安全和复现六类 Agent；六个可复用 Skill 分别完成约束结构化、数据来源核验、多目标排序、信息增益实验建议、人工审批与回滚、证据包审计。系统把每一步的输入输出摘要、证据引用、异常、权限和审批写入可回放 Trace，并在证据不足时主动阻断“发现”声明或物理执行。当前零依赖原型可在普通电脑离线运行，以电池正极筛选展示完整闭环；演示数据被强制标记为 illustrative，不冒充实验结果。后续将把同一能力迁移到配方、工艺和药物研发工作流，形成跨行业可复用的开源 Skill 与验证基准。

## 大奖叙事

项目不卖“又一个科研聊天机器人”，而是补齐 Agent 从建议走向企业研发执行最缺的控制面：证据、权限、审批、回滚、复现。材料只是第一个 Demo；Skill 契约、Agent Identity、Trace 与安全闸门可迁移到所有高风险知识工作。这一跨领域复用面对应全场大奖关注的开源影响力、复用价值与成长潜力。

## 绝不虚构

- 演示数据只证明工程闭环，不证明材料结论。
- 获得真实数据、科研合作者或实验结果后才更新相应表述。
